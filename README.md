## Installation

ServerAPI-SDK is available from [Jitpack](https://jitpack.io/#2b2tAustralia/ServerAPI-SDK):

### Gradle

```groovy
repositories {
    maven { url 'https://jitpack.io' }
}

dependencies {
    implementation 'com.github.2b2tAustralia:ServerAPI-SDK:Tag'
}
```
